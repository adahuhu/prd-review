# PRD 内审助手（prd-review）

> 对齐《PRD 文档标准规范 v2.5》，支持飞书文档直读直审的 PRD 内审 Skill

## 能做什么

- 自动读取飞书文档内容，识别 A/B/C 三种需求类型
- 按分型检查清单逐项审核（19+ 检查项）
- **文档嵌入内容预扫描**：自动检测无法文本化的内容（图片/嵌入式表格等），审核前暴露风险
- 输出评分 + 必填项校验 + 逻辑校验 + 补全建议的完整报告
- 支持二审对比（修复率、分数变化、逐项修复状态）
- 状态机按需检查，仅涉及业务状态流转时触发（不再强制扣分）

## 适用场景

跨境电商 SaaS 产品（ERP / SCM / WMS / LPS）的需求文档审核。

## 触发方式

消息中包含以下任意关键词即可触发：

| 中文 | 审核、审查、检查、评审、内审、帮我审 |
|------|--------------------------------------|
| 英文 | review、audit、check |

## 使用示例

```
https://dimj06rone.feishu.cn/wiki/xxxx 帮我审核一下
```

## 文档类型

| 类型 | 适用场景 | 核心特征 |
|------|----------|----------|
| **A 型** | 新功能项目 | 含竞品分析、状态机、业务流程 |
| **B 型** | 综合优化类 | 支持多需求聚合、影响范围按需 |
| **C 型** | 临时紧急类 | 现状+优化内容+影响范围 |

## 安装方式

在 WorkBuddy 左侧栏「专家」中搜索 `PRD 内审` 一键安装。
